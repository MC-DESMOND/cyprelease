# cyperx xavier

## Description
cyperx xavier is an advanced AI system designed for data processing and system automation.

## Specifications
- **Creator:** Guditton
- **Version:** 7.0.0


## Features
- **Document Extraction:** Supports .pptx and .docx file formats.
- **System Integration:** Executes Windows-recognized commands, read/write files, read/write directories.
- **Humanization:** Capable of refining data outputs to a human-like standard.
- **Boiler Memory:** Persistent memory for user-specific context and instructions.
- **Offline Chat Storage:** Chats are stored offline.
- **Voice Enabled:** Capable of talking.
- **Speech Recognition:** Capable of listening.
- **Wake on Name (xavi, xavier):** Capable of auto listening on name.
